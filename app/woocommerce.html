Переподключение хлебных крошек

  • отключить старые добавив хук в function.php (remove_action)
  • прикрепить к новому хуку ( add_action() )
  • инициализировать в теме с помощью функции do_action()

Вывод корзины

  • Вывод содержимого корзины черерз хук
    • создать функцию в function.php (add_action)
    • содержимое фнукции указать (или добавить напрямую в файл темы)
    • вывести в тему через хук (do_action)
  • Вывод счетчика корзины
    • добавить код в файл темы < div class="header-cart-count">cart->get_cart_contents_count(); ?>< /div>
    • прикрепить в function.php функцию к хуку фильтру подробнее

Описание структуры файлов папки woocomerce

  • content-single-product.php - настройка структуры страницы товара, можно задать col-6, изменять с помощью хуков
  • single-product.php - настройка страницы товара без основного контента (изображения, описание, табы), можно отключить сайд бар
  • content-product.php - настройка списка товаров в категории, можно задавать классы к тегу li или изменить его, доступ к контейнеру ul в loop/loop-start.php
  • archive-product.php - структура страница категории товаров (/shop), изменять с помощью хуков
